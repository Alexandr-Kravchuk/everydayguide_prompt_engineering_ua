# 📚 Словник термінів промпт-інжинірингу

##
<details>
<summary>Виберіть, щоб переглянути словник термінів.</summary>

| Термін | Визначення |
|--------|------------|
| 🎯 Промпт | Конкретний запит або інструкція, яку дають моделі ШІ, щоб отримати бажаний результат чи поведінку. |
| 🧠 Мовна модель (LM) | Система ШІ, натренована на великому обсязі текстових даних для розуміння й генерації мови, подібної до людської. |
| 🔗 Chain of Thought (CoT) | Техніка промптингу, яка заохочує покрокове міркування у відповідях ШІ. |
| 🚀 Zero-Shot | Здатність моделі ШІ виконувати завдання без надання конкретних навчальних прикладів. |
| 🎯 Few-Shot | Підхід, коли для спрямування відповіді ШІ надається кілька прикладів. |
| 🤔 ReAct | Метод промптингу, який поєднує міркування та здійснення дій для динамічного розв’язання задач. |
| 🌳 Tree of Thoughts (ToT) | Підхід, що одночасно досліджує кілька шляхів міркувань для складних задач. |
| 🔄 Self-Consistency | Техніка підвищення надійності, за якої генерується кілька незалежних рішень і обирається найбільш узгоджене. |
| 📄 Hypothetical Document Embeddings (HyDE) | Метод використання згенерованих гіпотетичних документів для покращення пошуку інформації та відповіді на запитання. |
| 🏗️ Least-to-Most Prompting | Стратегія, що розбиває складні задачі на простіші й керовані підзадачі. |
| 🔗 Prompt Chaining | Послідовне поєднання кількох промптів для виконання багатокрокових завдань. |
| 📊 Graph Prompting | Використання графових структур для відображення складних зв’язків у промптах. |
| 🔄 Recursive Prompting | Ітеративна техніка, яка використовує вихід моделі як вхід для наступних промптів. |
| 💡 Generated Knowledge | Використання інформації, згенерованої ШІ, щоб посилити відповіді у знаннєво інтенсивних задачах. |
| ⚙️ Automatic Reasoning and Tool-Use (ART) | Метод, що поєднує міркування з автоматичним вибором і використанням інструментів. |
| 🛠️ Automatic Prompt Engineer (APE) | Підхід до автоматичного створення та оптимізації промптів. |
| 🎚️ Налаштування тону та стилю | Прийоми, що дають змогу керувати стилем і тоном тексту у відповідях ШІ. |
| 🎭 Рольова модель | Призначення ШІ конкретної персони, щоб спрямувати стиль відповіді. |
| ⚖️ Етичні міркування | Важливі аспекти в промпт-інжинірингу, зокрема мінімізація упереджень і безпека контенту. |
| ✅ Оцінювання промптів | Аналіз ефективності промптів за такими критеріями, як релевантність, точність і зв’язність. |

</details>
