## 🔄 Рекурсивний промптинг

Рекурсивний промптинг — це техніка, за якої вихід моделі використовується як вхід для наступних промптів, утворюючи цикл зворотного зв’язку. Такий метод дає змогу ітеративно вдосконалювати відповіді, поступово створюючи дедалі складніший або детальніший контент. Особливо корисний для завдань, що виграють від поетапного покращення або поглибленого аналізу.

### Сфери застосування

<details>
<summary>Натисніть, щоб переглянути приклади застосування</summary>

1. **Поглиблення аналізу теми:** поступове дослідження складних питань із дедалі більшими нюансами.
2. **Генерація дедалі складнішого контенту:** створення детальніших та відшліфованих матеріалів.
3. **Самовдосконалювані системи:** розробка відповідей, які спираються на попередні ітерації.

</details>

### Приклад структури

```markdown
Завдання: "Напиши коротке оповідання про загадковий предмет"

Ітерація 1:
Промпт: "Склади односкладне оповідання про загадковий предмет"
Вихід: "У лісі з’явилася світна куля, дивуючи всіх, хто її бачив."

Ітерація 2:
Промпт: "Розшир попереднє речення до трьох речень, додавши деталі про кулю та її вплив"
Вихід: "У лісі з’явилася світна куля, дивуючи всіх, хто її бачив. Її пульсуюче світло гіпнотизувало тварин і людей, притягуючи все ближче. Коли стемніло, куля почала ширяти, відкидаючи моторошні тіні між деревами."

Ітерація 3:
Промпт: "На основі попереднього абзацу напиши оповідання приблизно на 100 слів, додавши персонажа, який досліджує кулю"
Вихід: [Оповідання на 100 слів, що розвиває сюжет]
```

### Приклади рекурсивного промптингу

<details>
<summary>Приклад 1: Індивідуальна навчальна програма</summary>

```markdown
1. "Визнач широку сферу, яку хочеш опанувати (наприклад, дата-сайєнс, діджитал-маркетинг, креативне письмо)."
2. "Для сфери [Назва галузі] переліч 5–7 ключових тем або навичок, які потрібно опанувати."
3. "Для теми [Назва теми] створіть навчальний план із ключовими поняттями та підтемами."
4. "Розроби детальний план вивчення [Назва поняття], включаючи ресурси, вправи та проєкти."
5. "Удоскональ план, додавши рівні складності, орієнтовний час і контрольні точки."
6. "Переглянь усю програму для [Назва галузі] і запропонуй покращення щодо послідовності та практичного застосування."

Повторюй кроки 3–6 для кожної теми, доки не отримаєш повну персоналізовану навчальну програму.
```

</details>

<details>
<summary>Приклад 2: Стратегія створення контенту з підтримкою ШІ</summary>

```markdown
1. "Визнач цілі контент-маркетингу та цільову аудиторію для конкретної платформи (наприклад, блог, YouTube, подкаст)."
2. "Згенеруй 10 загальних ідей контенту, що відповідають цілям і аудиторії."
3. "Для [Ідея контенту] створіть детальний план із основними пунктами, підзаголовками та ключовими висновками."
4. "Розкрий [Основний пункт], навівши конкретні приклади, дані або історії."
5. "Удоскональ контент, порадивши інструменти ШІ для дослідження, написання, редагування чи створення візуальних матеріалів для кожного розділу."
6. "Переглянь увесь матеріал і запропонуй способи оптимізації під SEO, залучення аудиторії та алгоритми платформи."

Повторюй кроки 3–6 для кожної ідеї, поки не сформуєш комплексну стратегію зі складовими, що підтримуються ШІ.
```

</details>

<details>
<summary>Приклад 3: Проєктування системи розумного дому</summary>

```markdown
1. "Переліч основні зони будинку, які хочеш автоматизувати (наприклад, освітлення, безпека, клімат)."
2. "Для зони [Назва зони] визнач 3–5 сценаріїв або функцій автоматизації."
3. "Розроби детальний план для [Назва сценарію], вказавши потрібні пристрої, підключення та базову логіку."
4. "Удоскональ план автоматизації, додавши розширені тригери, умови та дії для точнішого керування."
5. "Запропонуй інтеграції зі ШІ (наприклад, голосові асистенти, прогностичні алгоритми) для покращення сценарію."
6. "Переглянь всю систему розумного дому й порадь, як підвищити енергоефективність, зручність і масштабованість."

Повторюй кроки 2–6 для кожної зони, доки не отримаєш комплексну схему автоматизації.
```

</details>

## 💡 Поради для ефективного рекурсивного промптингу

1. **Починайте просто:** стартуйте з базового промпту й поступово ускладнюйте.
2. **Будьте конкретними:** давайте чіткі інструкції для кожної ітерації, щоб спрямувати вдосконалення.
3. **Спирайтеся на попередні результати:** явно згадуйте або використовуйте елементи з попередніх відповідей.
4. **Чергуйте фокус:** поєднуйте розширення контенту з уточненням деталей.
5. **Визначайте мету:** окреслюйте, чого хочете досягти на кожному етапі.
6. **Відстежуйте прогрес:** регулярно оцінюйте, чи дає рекурсія помітні покращення.
7. **Знайте міру:** встановлюйте межу кількості ітерацій, щоб уникнути спадної віддачі.

<details>
<summary>📝 Практичне завдання: розроблення стратегії рекурсивного промптингу</summary>

1. Оберіть складне завдання або проєкт, що виграє від ітеративного вдосконалення (наприклад, написання дослідницької роботи, розроблення бізнес-плану, створення комплексної фітнес-програми).

2. Визначте загальну мету вашої стратегії рекурсивного промптингу.

3. Складіть початковий промпт, який охоплює найпростіший аспект завдання.

4. Розробіть серію з 5–7 наступних промптів, які поступово:
   - розширюють контент;
   - додають деталі чи глибину;
   - вдосконалюють попередні результати;
   - вводять нові елементи або точки зору.

5. Для кожного промпту серії:
   - поясніть його призначення та зв’язок із попередніми ітераціями;
   - опишіть очікуваний тип результату;
   - запропонуйте, як будете використовувати або оцінювати цей результат.

6. Визначте потенційні проблеми у вашій стратегії рекурсивного промптингу та запропонуйте рішення.

7. Опишіть, звідки зрозумієте, що досягли задовільного фінального результату.

8. Поміркуйте, чому такий рекурсивний підхід може бути ефективнішим, ніж один універсальний промпт.

</details>
