## 🤔 ReAct (Reasoning and Acting)

ReAct (Reasoning and Acting) — це просунута техніка промптингу, яка поєднує міркування з виконанням дій. Вона дозволяє моделі взаємодіяти з оточенням або зовнішніми інструментами під час розв’язання задачі, забезпечуючи динамічний і адаптивний процес. Такий підхід імітує людський стиль вирішення проблем, чергуючи фази «подумати» й «зробити».

### Сфери застосування

<details>
<summary>Натисніть, щоб переглянути приклади застосування</summary>

- **Інтерактивне розв’язання задач:** дозволяє оперативно реагувати на нову інформацію.
- **Пошук і синтез інформації:** допомагає збирати дані з різних джерел та опрацьовувати їх.
- **Прийняття рішень у динамічному середовищі:** сприяє адаптивним рішенням у змінних умовах.

</details>

### Приклад структури

Працюючи за методом ReAct, чергуйте «думки» й «дії»:

```markdown
Thought: [Початкове міркування]
Action: [Дія або використання інструменту]
Observation: [Результат дії]
Thought: [Наступний крок на основі спостереження]
```

### Приклади промптів ReAct

Нижче наведено кілька сценаріїв використання ReAct (Reason + Act):

<details>
<summary>Оптимізація роботи служби підтримки</summary>

```markdown
**Промпт:** Застосуймо метод ReAct, щоб оптимізувати процес обслуговування клієнтів:

> Thought: Спершу потрібно проаналізувати поточні показники.
> Action: Запитай у CRM середній час відповіді, відсоток вирішених звернень і індекс задоволеності за останній квартал.
>
> Observation: Середній час відповіді — 4 години, вирішення — 85%, задоволеність — 7,5/10.
>
> Thought: Ці показники можна поліпшити. Перевіримо відгуки клієнтів на типові скарги.
> Action: Проаналізуй останні опитування клієнтів на повторювані теми.
>
> Observation: Багато хто скаржиться на довгі очікування й необхідність повторно пояснювати проблему кільком агентам.
>
> [Продовжуйте наступні кроки...]

Тепер застосуйте ReAct, щоб оптимізувати інший аспект бізнесу, наприклад, управління запасами чи онбординг співробітників.
```

</details>

<details>
<summary>Планування відпустки</summary>

```markdown
**Промпт:** Скористаймося методом ReAct для планування відпустки:

> Thought: Почнемо з порівняння можливих напрямків.
> Action: Переглянь туристичні сайти й підбери напрямки в нашому бюджеті з відповідним кліматом.
>
> Observation: Підходять три варіанти: Балі, Хорватія та Коста-Рика.
>
> Thought: Порівняймо вартість перельотів у ці країни.
> Action: Перевір ціни на квитки на обрані дати через агрегатори.
>
> Observation: Квитки на Балі найдорожчі, а до Хорватії — найвигідніші.
>
> [Продовжуйте наступні кроки...]

Тепер використайте ReAct для планування іншого типу подорожі, наприклад, автотуру чи вікенду в місті, спираючись на актуальні дані.
```

</details>

<details>
<summary>Персональний план фітнесу й харчування</summary>

```markdown
**Промпт:** Застосуймо ReAct для створення персонального плану фітнесу й харчування:

> Thought: Спершу оцінимо поточний стан здоров’я.
> Action: Введи вік, зріст, вагу та активність у калькулятор BMI.
>
> Observation: ІМТ становить 27,5 — це категорія «надмірна вага».
>
> Thought: Визначимо конкретні фітнес-цілі.
> Action: Використай шаблон SMART, щоб сформулювати цілі.
>
> Observation: Ціль: схуднути на 10% за 6 місяців і пробігти 5 км.
>
> [Продовжуйте наступні кроки...]

Тепер використайте ReAct, щоб розробити план оздоровлення для конкретного стану, наприклад, гіпертонії чи діабету, спираючись на авторитетні медичні ресурси.
```

</details>

<details>
<summary>План кар’єрного розвитку</summary>

```markdown
**Промпт:** Скористаймося ReAct, щоб створити план професійного зростання:

> Thought: Спершу оцінимо поточні навички й виявимо прогалини.
> Action: Пройди онлайн-тест на навички, релевантні галузі.
>
> Observation: Тест показав сильні лідерські навички, але потребу розвинути аналітику даних.
>
> Thought: Дослідимо, які навички найбільш затребувані в галузі.
> Action: Переглянь вакансії й галузеві звіти щодо ключових навичок.
>
> Observation: Попит високий на візуалізацію даних і машинне навчання.
>
> [Продовжуйте наступні кроки...]

Тепер застосуйте ReAct, щоб створити план розвитку для іншої мети, наприклад, переходу в нову сферу чи запуску власної справи, використовуючи реальні дані ринку праці та навчальні ресурси.
```

</details>

### 💡 Поради для ефективних промптів ReAct

1. **Чітко формулюйте дії:** вказуйте, яку саме дію потрібно виконати або який інструмент використати.
2. **Робіть спостереження реалістичними:** надавайте правдоподібні результати, щоб зберігати логіку процесу.
3. **Заохочуйте різні підходи:** пропонуйте моделі розглянути альтернативи чи інші ракурси.
4. **Ітеруйте та адаптуйтеся:** використовуйте спостереження, щоб визначити наступні кроки.
5. **Баланс глибини та широти:** досліджуйте кілька аспектів задачі, водночас приділяючи увагу деталям.
