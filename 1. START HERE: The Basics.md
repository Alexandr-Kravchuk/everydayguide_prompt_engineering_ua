## Ключові елементи дизайну промптів

<img src="key_elements_prompt_design.png" alt="Alt text" width="350" height="300">

- ✨ **Чіткість**: формулюйте інструкції конкретно й однозначно.
- 🏛️ **Контекст**: надавайте релевантну фонову інформацію.
- 🚧 **Обмеження**: задавайте рамки для відповіді ШІ.
- 🧪 **Приклади**: за можливості додавайте зразки вхідних і вихідних даних.
- 🗂️ **Формат**: визначайте бажану структуру відповіді.


## 📌 Поширені техніки промптингу

### Виберіть випадаючий список, щоб розгорнути перелік популярних технік промптингу 👇🏽
<details>
<summary>Техніки промптингу</summary>

- 🧠 **Chain of Thought (CoT)**: покращує міркування завдяки проговоренню проміжних кроків.
- 🚀 **Zero-Shot Chain of Thought (Zero-Shot-CoT)**: застосовуйте CoT без попередніх прикладів чи навчання.
- 🎯 **Few-Shot Chain of Thought (Few-Shot-CoT)**: використовуйте приклади, щоб спрямувати хід думок.
- 🤔 **ReAct (Reasoning and Acting)**: поєднує міркування з дією для кращих відповідей.
- 🌳 **Tree of Thoughts (ToT)**: ієрархічно впорядковує думки для ефективніших рішень.
- 🔄 **Self-Consistency**: забезпечує сталі та послідовні відповіді на запити.
- 📄 **Hypothetical Document Embeddings (HyDE)**: застосовує вбудовування для представлення можливих документів.
- 🏗️ **Least-to-Most Prompting**: починайте з простого й поступово ускладнюйте.
- 🔗 **Prompt Chaining**: поєднуйте кілька промптів у послідовну історію.
- 📊 **Graph Prompting**: використовуйте графові структури для відображення складних взаємозв’язків.
- 🔄 **Recursive Prompting**: ітеративно вдосконалюйте промпти для кращих результатів.
- 💡 **Generated Knowledge**: використовуйте згенерований контент для подальших міркувань.
- ⚙️ **Automatic Reasoning and Tool-Use (ART)**: автоматизуйте процеси міркування та взаємодії з інструментами.
- 🛠️ **Automatic Prompt Engineer (APE)**: інструменти для автоматичної генерації та доопрацювання промптів.
